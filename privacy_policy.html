8. Content Security Policy (CSP)

To further protect user security, the EchoRec extension implements a strict Content Security Policy (CSP). This policy restricts which sources can be used for scripts, styles, media, and other content.

Specifically:

  • Scripts are restricted to extension-only sources (no external JS execution).
  • Styles are limited to local and inline-safe rules, with no external loading from untrusted sources.
  • No remote assets (e.g., fonts, analytics, or images) are loaded from external URLs.

This ensures that malicious code injection or data leakage is strongly mitigated, helping maintain a safe recording environment for all users.
